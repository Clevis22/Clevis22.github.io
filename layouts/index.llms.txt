# TinyWeights.dev

> {{ site.Params.description }}

TinyWeights.dev is an independent blog focused exclusively on small language models (roughly 0.3B to 35B parameters) that run on local hardware: laptops, desktops, single-board computers, and phones. Coverage includes hands-on deployment guides, quantization explainers, benchmark comparisons, and hardware sizing. Posts cite official model cards, technical reports, and papers as primary sources, and include exact commands, measured file sizes, and RAM/VRAM requirements.

## Tools

- [Which model fits your hardware?](https://tinyweights.dev/fit/): Interactive wizard and VRAM/RAM calculator. Enter your memory and use case, get fitting models with recommended GGUF quantization.
- [Try a small LLM in your browser](https://tinyweights.dev/try/): Live WebGPU chat demo running quantized models (SmolLM2, Qwen2.5, Llama 3.2) entirely client-side via WebLLM.
